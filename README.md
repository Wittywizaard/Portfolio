# Tanya's Portfolio

A modern, interactive portfolio website built with Next.js, TypeScript, and Tailwind CSS. Features smooth animations with Framer Motion and is optimized for deployment on Vercel.

## Features

- Responsive design
- Smooth scrolling navigation
- Animated sections with Framer Motion
- Professional layout showcasing experience, skills, projects, and education
- Contact information and social links

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment to Vercel

1. Push your code to a GitHub repository
2. Go to [Vercel](https://vercel.com) and sign in
3. Click "New Project" and import your GitHub repository
4. Vercel will automatically detect it's a Next.js project and deploy it
5. Your portfolio will be live at a Vercel domain (e.g., `your-project.vercel.app`)

## Technologies Used

- Next.js 16
- TypeScript
- Tailwind CSS
- Framer Motion
- React 19

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
